# SupremeBot

## Información

SupremeBot es un bot de Moderación, Utilidades, Leveling y mucho más que lo hace perfecto para tu servidor!

## Características

- Sistema de Moderación
- Almacenamiento de Sanciones mediante bases de datos
- Leveling con Autorol
- Prefix Customizable
- Logs
- Online 24/7
